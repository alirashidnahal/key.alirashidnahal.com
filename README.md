# PGP Key

A minimal, static web page that provides easy access to my PGP public key.

## Purpose

This page allows visitors to:
- **Download** my PGP public key directly.
- **Fetch** the key from trusted public keyservers:
  - [keys.openpgp.org](https://keys.openpgp.org)
  - [keyserver2.pgp.com](https://keyserver2.pgp.com)

## Live Demo

🔗 [key.alirashidnahal.com](https://key.alirashidnahal.com)

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla)

## Why?

Sharing a PGP public key via a dedicated, HTTPS-secured URL makes it easier for others to verify my identity and encrypt communications.
